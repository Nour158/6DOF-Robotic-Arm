# 6-DOF Robotic Arm

A robotics project that controls a 6-DOF robotic arm using Arduino and servo motors.

## Features

- Forward Kinematics
- Inverse Kinematics
- Servo motor control
- End-effector positioning

## Technologies

- Arduino
- C++
- Servo Library
- Robotics
- Forward Kinematics
- Inverse Kinematics
